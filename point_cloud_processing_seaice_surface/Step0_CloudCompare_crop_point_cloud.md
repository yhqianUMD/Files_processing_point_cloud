# Steps to crop a point cloud with CloudCompare software

1. Load the point cloud to CloudCompare
2. Select the loaded data, and then click the segment sign "scizzer"
3. Use the rectangle tool to crop the point cloud
4. Click Segment in or out
5. To save or export the cropped point cloud, go to File --> Save. To save the points in 4 digits after the decimal point, set "Coordinates Precision" as 4 and "Scalar Precision" as 4, while leaving the others as the default values.
